# HxA Macalester Website - Editor Guide

This guide is for non-technical editors who need to update website content.

## Getting Started

### Accessing the Editor

1. Go to **hxamac.org/admin** (or your-site.netlify.app/admin before domain setup)
2. Click "Login with Netlify Identity"
3. Enter your email and password
4. You'll see the content management dashboard

### First Time Setup

If you haven't logged in before:
1. You'll receive an email invitation from Netlify
2. Click the confirmation link
3. Set your password
4. You can now log in at /admin

---

## Editing Content

### Leadership / Team Members

To add or edit team members:

1. Go to **Leadership** in the sidebar
2. Click an existing entry to edit, or **New Leadership** to add
3. Fill in the fields:
   - **Name**: Full name
   - **Role**: e.g., "Co-Chair", "Faculty Advisor"
   - **Department**: Academic department (optional)
   - **Bio**: Short biography
   - **Email**: Contact email
   - **Photo**: Upload a headshot (optional)
   - **Order**: Lower numbers appear first
   - **Type**: Select from Co-Chair, Faculty Advisor, or Member

4. Click **Save** to save as draft, or **Publish** to make live

### Events

To manage events:

1. Go to **Events** in the sidebar
2. Click an existing event or **New Events**
3. Fill in the fields:
   - **Title**: Event name
   - **Date**: Select from calendar
   - **Time**: e.g., "6:00 PM - 8:00 PM"
   - **Location**: Building and room
   - **Description**: Event details
   - **Link**: URL for registration or more info (optional)
   - **Past Event**: Toggle on for events that have already happened

### Resources

To add recommended readings or resources:

1. Go to **Resources** in the sidebar
2. Click an existing resource or **New Resources**
3. Fill in the fields:
   - **Title**: Book/article title
   - **Author**: Author name
   - **Description**: Brief description
   - **Link**: URL to resource
   - **Category**: Book, Article, Video, Tool, or External Resource
   - **Order**: Lower numbers appear first

## Publishing Workflow

### Draft vs Published

- **Save**: Saves your changes as a draft (not visible on site)
- **Publish**: Makes changes live on the website

### Editorial Workflow

The site uses an editorial workflow with these stages:

1. **Drafts**: Work in progress, not visible
2. **In Review**: Ready for someone to check
3. **Ready**: Approved and ready to publish

You can move items between stages using the **Set status** dropdown.

## Uploading Images

### Adding Images

1. When editing content, click the image field
2. Choose "Upload" to upload a new image
3. Or "Choose an existing image" to use one already uploaded

### Image Guidelines

- **Headshots**: Square aspect ratio, minimum 400x400px
- **File types**: JPG, PNG, or WebP
- **File size**: Under 500KB for fast loading
- **Naming**: Use descriptive names like `john-smith-headshot.jpg`

## Tips for Good Content

### Writing Event Descriptions

- Start with what the event is about
- Include who should attend
- Mention any prerequisites or preparation needed
- Keep it under 100 words

### Team Member Bios

- Write in third person ("Matthew studies..." not "I study...")
- Keep to 2-3 sentences
- Focus on relevance to HxA mission
- Include year and major if student

## Troubleshooting

### Can't Log In

- Check that you're using the correct email
- Try "Forgot password" if needed
- Contact the site administrator if issues persist

### Changes Not Showing

- Make sure you clicked "Publish", not just "Save"
- Clear your browser cache
- Changes can take 1-2 minutes to appear after publishing

### Image Upload Failed

- Check file size is under 5MB
- Try a different file format (JPG usually works best)
- Make sure you have a stable internet connection
